# Промышленные посадочные страницы

Три адаптивные страницы на общем шаблоне:

- `/test-stands/` — испытательные стенды;
- `/hydraulic-stations/` — гидростанции;
- `/hydraulic-cylinders/` — гидроцилиндры.

## Запуск

```powershell
rtk npm run dev
```

Открыть `http://127.0.0.1:4173`.

## Настройка перед публикацией

В `config.js` задаются название компании, телефон, email, адрес и URL обработчика формы (`formEndpoint`).

В карточках производства и кейсов предусмотрены явные места для реальных фотографий и подтвержденных материалов. Их необходимо заменить перед публикацией. Форма отправляет `multipart/form-data` на указанный endpoint и поддерживает вложение до 20 МБ.
